# Install

### 1. Setup environment variables as described in `Development`.

* [Development Section](/doc/DEVELOPMENT.md)

### 2. Install dependencies

```
$ npm install
```

### 3. Start the server

```
// for development
$ npm run start-dev 

// for production
$ npm build // create js build from ts
$ npm start
```
